# 四、下载完之后，解压，把文件夹里面的`chromedriver.exe`复制到你`项目的python.exe同级目录`下

# 五、在你项目环境里面，输入以下命令即可

```shell
pip install selenium -i http://pypi.douban.com/simple --trusted-host pypi.douban.com
```

`然后就可以愉快的玩耍啦！！！`
